# uniapp 仿网易云音乐播放器

后端使用的是 github 开源的 API：[Binaryify/NeteaseCloudMusicApi: 网易云音乐 Node.js API service (github.com)](https://github.com/Binaryify/NeteaseCloudMusicApi)

主要技术栈：uni-app、vue2、vuex、scss

本地克隆完项目，启动完后端后，直接在 HbuilderX 中启动即可
